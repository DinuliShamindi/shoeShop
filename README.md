👟 SHOE Store Website

A modern and responsive front-end e-commerce website for showcasing and selling shoes. Built using HTML, CSS, and a little JavaScript, this project demonstrates a complete UI for an online shoe store.

🌐 Live Preview
(https://dinulishamindi.github.io/shoeShop/)

📌 Features
🏠 Beautiful landing (Home) section
🛍️ Product listing grid with cards
⭐ Customer review section
👟 Interactive image gallery (About section)
🚚 Services section (delivery, support, etc.)
🔐 Login form UI
📱 Responsive layout (depends on CSS)
🎨 Font Awesome icons integration

🛠️ Technologies Used
HTML5
CSS3
JavaScript (basic DOM manipulation)
Font Awesome (icons)

🛒 Product Cards

Each product contains:
Image
Title
Price
Star rating
Add to cart button (UI only)
